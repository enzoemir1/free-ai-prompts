## How to Use

1. Browse the [`prompts/`](./prompts) folder
2. Open any category file
3. Copy a prompt
4. Paste into ChatGPT, Claude, Gemini, or Midjourney
5. Replace the `[PLACEHOLDERS]` with your specific details

Each prompt includes tips for better results.

---

## Example

From the **Marketing** pack:

```
Write a concise 2-email follow-up sequence for a cold outreach about [PRODUCT]
to [PROSPECT_NAME] at [COMPANY]. Include:
1) Subject line (max 50 chars)
2) Opening line referencing [RECENT_EVENT] or [PAIN_POINT]
3) Value proposition in 2 sentences
4) Clear CTA to schedule a 15-min call

Keep each email under 150 words.
```

💡 **Tip:** Focus on personalization and a single clear CTA.
