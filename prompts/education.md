### Prompt 1

```
Generate a lesson plan for **[Subject]** for **[Grade Level]** focusing on **[Topic]**. Present the plan in markdown with the following sections: **Objectives**, **Materials**, **Introduction (5 min)**, **Main Activity (30 min)**, **Closure (5 min)**, **Assessment**, and **Differentiation**. Use bullet points for each item and include a time allocation for every activity.
```

💡 **Tip:** Keep language simple and stick to one main learning objective.

### Prompt 9

